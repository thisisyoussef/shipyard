# Tic-Tac-Toe Test Target

This directory is the dedicated local Shipyard target for greenfield testing.
It is intentionally minimal so Shipyard can scaffold the project from scratch.

## Product Brief

Build an interactive tic-tac-toe game for the browser.

## Current State

- No application code is checked in yet.
- `AGENTS.md` captures the target-specific defaults for Shipyard.
- The workspace helper `scripts/init-test-target.sh` can initialize this
  directory as a local Git repository on `main`.
